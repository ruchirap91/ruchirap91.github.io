The ternary operator isn't meant to execute an action but rather to yield a value.
return is a statement. It does not evaluate to anything. We are using a ternary operator, which has the syntax - condition ? expression1 : expression2
and evaluates to expression1 if the condition holds, and expression2 otherwise. This implies that expression1 and expression2 must themselves evaluate to something, and cannot be statements.

Question 6: Demonstrate difference between function declaration and expression

The difference lies in how the browser loads them into the execution context.
Function declarations load before any code is executed(since they get hoisted).
Function expressions load only when the interpreter reaches that line of code.
So if you try to call a function expression before it's loaded, you'll get an error.
If you call a function declaration instead, it'll always work, because no code can be called until all declarations are loaded.
Function expression:

alert(exprTest()); // ERROR! exprTest is not a function
var exprTest = function() { return 5; }

Function declaration:

alert(decTest()); // Alerts 5. Declarations are loaded before any code can run.
function decTest() { return 5; }

Question 9: WAF: To check if palindrome

function palindrome(str) { // declaring a function palindrome with argument str
var len = str.length; //check length of the string passed
for ( var i = 0; i < Math.floor(len/2); i++ ) { // in palindrome, apart from center letter(if present), other letters are exactly same at their particlular position, hence we only need to loop through half the length of the string
if (str[i] !== str[len - 1 - i]) { // here we check if the ith index matches with the same index from the other end, eg, in malayalam, we check if m matches with len-i-1 = 9-0-1 = 8 = m
return false; // if it does not match, it is not palindrome so we return false
}
}
document.write("palindrome"); // if it does not satisfy if statement, it implies that it is a palindrome. note that all letters are checked in the if condition since the if is within a for loop.
}
palindrome("malayalam"); //output: palindrome

fib[0] = 0;// every number in fibo series is a sum of prev 2 numbers. to start we initially need to declare 0 and 1, since their addition will kickstart the fibo sequence
fib[1] = 1;
document.write(fib[1]); // printing 1
for(i=2; i<=j; i++)
{
// Next fibonacci number = previous + one before previous

}
