Question 1
var counter = 0; counter++; ++counter; alert( counter ); What will alert this and why?
Step1: counter++; At this step, the count value will be 0, it will increment after this step, it being post-increment operator
Step2: ++counter; At this point the count value is 1, and since this is a pre-increment operator, the count will now be 2
Step3: alert( counter ); This will give a value of 2
